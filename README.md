# 🎬 Netflix / OTT Data Analysis Dashboard

## 📊 Project Overview

This project is an interactive **Netflix / OTT Data Analysis Dashboard** built using Python and Streamlit.
It analyzes Netflix content data to extract meaningful insights about movies, TV shows, genres, countries, and trends.

---

## 🚀 Features

* 📺 Interactive Dashboard using Streamlit
* 🎯 Filters (Content Type, Country)
* 📊 Visualizations (Bar Charts, Line Charts)
* 🎬 Netflix-style UI (Dark Theme + Cards)
* 📈 Insights on content trends

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas
* Matplotlib
* Seaborn
* Streamlit

---

## 📂 Dataset

* Netflix Movies & TV Shows Dataset
* Includes:

  * Title
  * Type (Movie / TV Show)
  * Country
  * Release Year
  * Genre
  * Rating

---

## 📊 Key Insights

* 🎬 Movies dominate the platform compared to TV Shows
* 🌍 United States produces the highest content
* 🎭 International Movies is the most popular genre
* ⭐ TV-MA is the most common rating
* 📅 Content growth increased significantly after 2015

## ▶️ How to Run the Project

### Step 1: Clone the repository

git clone https://github.com/Rudra98341/Netflix-OTT-Data-Analysis-Dashboard.git

### Step 2: Install dependencies
pip install -r requirements.txt


### Step 3: Run the dashboard
streamlit run app.py
